ClearOS

ClearOS is a free and open source cloud-connected server, network, and gateway operating system and it is designed for small organizations and distributed environments.If your XMPP client supports the standard, you should be able to join the room, by clicking here If that doesn't work, add conference.wikisuite.net as a conference server, and then joint the clearos room.

Openfire

Openfire Meetings is a real-time communication server implementing the XMPP (Jabber) and WebRTC protocols with a SIP gateway It offers a very large number of features and a great admin panel and source code is available under the Apache2 license. It includes Jitsi Meet and Converse, and interacts with Pàdé.

xmpp:open_chat@conference.igniterealtime.org?join

Converse.js

Converse is a free and open-source XMPP chat client in your browser.WikiSuite and Converse are working closely together. Notably to convert v4 to Bootstrap Converse offers both pop up chat and full page chat. Converse is the built-in (but optional)XMPP client in Tiki Wiki CMS Groupware: https://doc.tiki.org/XMPP

xmpp:discuss@conference.conversejs.org?join

Pàdé

Pàdé (the "p" is pronounced explosively) is the Yoruba word for "Meet".Pàdé is a unified communications web browser extension for Openfire Meetings. It embeds Converse.Pàdé Presentation by WikiSuite
