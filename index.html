XMPP it's a federated protocol which WikiSuite has been use on many project like ClearOs conversejs OpenFire and others if youwant to use it you need an account but If you don't have an account you can just use your existing account If you don't have an account you can create a free one at conversejs.org

To have your own firstname.lastname@wikisuite.net XMPP account, you need to ask an admin (such as Fabio or Serge) to create an account for you.
